Interface JSONSchema

A subset of JSON Schema according to 2020-12 JSON Schema draft.

Represents a subset of a JSON Schema object that can be used by Gemini API. The difference between this interface and the Schema interface is that this interface is compatible with OpenAPI 3.1 schema objects while the types.Schema interface
